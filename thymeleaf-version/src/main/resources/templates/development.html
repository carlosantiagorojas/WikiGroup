PARA EL DESARROLLO

Endpoints para Administrador {/admin}

POST
/add Agrega un nuevo administrador a la base de datos.
GET
/all Obtiene todos los administradores de la base de datos.
GET
/{id} Obtiene un administrador por su ID.
PUT
/update/{id} Actualiza un administrador en la base de datos.
DELETE
/admin/delete/{id} Elimina un administrador de la base de datos por su ID.

Endpoints para Géneros {/genre}

POST
/add Agrega un nuevo género a la base de datos.
POST
/addMany Agrega varios géneros a la base de datos.
GET
/all Obtiene todos los géneros de la base de datos.
GET
/{id} Obtiene un género por su ID.
GET
/name/{name} Obtiene un género por su nombre.
PATCH
/patch/{id} Actualiza parcialmente un género en la base de datos.
PUT
/update/{id} Actualiza un género en la base de datos.
DELETE
/delete/{id} Elimina un género de la base de datos por su ID.

Endpoints para Relaciones entre Listas de Reproducción y Canciones {/playlistHasSong}

POST
/add/{playlistId}/{songId} Agrega una nueva relación entre lista de reproducción y canción a la base de datos.
GET
/all Obtiene todas las relaciones entre listas de reproducción y canciones de la base de datos.
GET
/get/{playlistId}/{songId} Obtiene una relación entre lista de reproducción y canción por sus IDs.
DELETE
/delete/{playlistId}/{songId} Elimina una relación entre lista de reproducción y canción de la base de datos por sus IDs.
