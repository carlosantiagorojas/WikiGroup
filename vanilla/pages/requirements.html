Historias de usuario

Requerimientos de los usuario votante y administrador para el proyecto

Historias de usuario principales para el usuario votante

Creación de cuenta

Creación de cuenta

Como usuario votante, quiero poder crear una cuenta en la plataforma para acceder a todas sus funcionalidades.

Activación de cuenta

Activación de cuenta

Como usuario votante, después de crear mi cuenta, quiero recibir un correo electrónico de activación con un enlace o código para confirmar y activar mi cuenta.

Autenticación de cuenta

Autenticación de cuenta

Como usuario votante registrado, quiero poder iniciar sesión en la plataforma utilizando mi correo electrónico y contraseña para acceder a mi perfil y utilizar las funcionalidades disponibles.

Ver listas por género musical

Ver listas por género musical

Como usuario votante, quiero poder explorar listas de canciones organizadas por género musical para descubrir nuevas canciones y listas populares.

Buscar canción

Buscar canción

Como usuario votante, deseo tener la capacidad de buscar una canción específica en la plataforma utilizando criterios de búsqueda como el nombre de la canción o el artista.

Votar/Desvotar

Votar/Desvotar

Como usuario votante, quiero tener la opción de votar positiva o negativamente por una canción en una lista y también poder cambiar mi voto si cambio de opinión.

Historias de usuario principales para el usuario administrador

Autenticación de usuario

Autenticación de usuario

Como usuario administrador, quiero poder iniciar sesión en la plataforma utilizando mis credenciales de administrador para acceder a las funciones de administración.

CRUD Generos musicales

CRUD Generos musicales

Como usuario administrador, necesito la capacidad de realizar operaciones CRUD (Crear, Leer, Actualizar y Eliminar) en géneros musicales. Esto incluye la creación de nuevos géneros, la edición de géneros existentes y la eliminación de géneros obsoletos.

CRUD Canciones

CRUD Canciones

Como usuario administrador, necesito poder gestionar las canciones en la plataforma, lo que implica la capacidad de realizar operaciones CRUD en las canciones. Esto incluye agregar nuevas canciones, editar información de canciones existentes y eliminar canciones que ya no son relevantes.
